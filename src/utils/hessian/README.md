# 重写hessian中间件

[增强功能]
```
1、支持中文（源代码中文乱码）。字符集必须是utf8。
2、支持double类型
3、支持字符串“000”
4、float支持不好，尽量使用字符串或double。如果须使用float,满足下列条件：
    1）服务端使用Float.parseFloat(String)
    2）约定小数位数，接收到数据后，指定位数进行四舍五入。使用num.toFix(精度)
5、map支持不够直观，调试或打印看不到数据。但是get能够获取到正确数据
6、Date类型支持不好，应该是时区不一致，少8小时。long值是对的，打印出来少8小时
```

[依赖]
```
,"long": "~3.2.0"
,"buffer-builder": "~0.2.0"
```
